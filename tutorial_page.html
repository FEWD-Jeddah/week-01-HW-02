Changing the color theme

You can modify the colors in VS Code to suit your preferences.

  1. In VS Code go to: File > Preferences > Color Theme.
  2. Click on the theme you like.

Moving the currnet line

You can move the current highlighted line up or down.

  1. Use this shortcut:alt + ↑ to move the line up.
  2. Use this shortcut:alt + ↓ to move the line down.

Inserting a new line

You can insert a line above or below the cursor.

  1. Use this shortcut:ctrl + Enter to insert a line below.
  2. Use this shortcut:ctrl + shift + Enter to insert a line above.

Auto Save

Just activate this feature and save yourself.

  • In VS Code go to: File > Auto Save.
